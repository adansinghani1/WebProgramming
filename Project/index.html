Young's Double Slit

The double slit experiment was brought in 1801 by the physicist Thomas Young.
It was used to prove one of the biggest debates of physics at the time, the behavior of light.
In this experiment we set out to reproduce the results once done in 1801 with the interference patterns.
The experiment is set up to have light beam coming from one end of the instrument where it is sent through a double slit which the slits are (a) apart to where it passes through a screen at a distance (l).
the double slit splits the beam into two. Once it hits the screen it a certain point will have either constructive or destructive interference.
Using the micrometer, we can move it to create interference patterns and read the distance. Using the oscilloscope, we can see the interference patterns but also measure the signal intensity. For its position we started at 0.1 mm all the way to 8.1 mm recording data at every 0.1 mm.

Hall Effect

The Hall effect was discovered in 1879 Edwin Hall.
Using this experiment, he was able to generate a electric current through a conductor with a magnetic field thus creating the electric field or the Hall field.
For this experiment we are going to recreate the experiment then taking data from changing variables of magnetic field, current, and hall voltage.
Using this data, we can find the chart density of the electric field.
The principle of the hall effect is that when a conductor is introduced to charged electrons or current that flows in one direction can be introduced to a perpendicular magnetic field.
We can then measure the hall voltage at right angles to the path.
